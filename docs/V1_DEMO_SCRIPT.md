# AI Governance Sentinel - V1 Demo Script

## Goal

Show that AI Governance Sentinel is not a chatbot, but an enterprise governance platform for AI agents.

## Demo flow

1. Open the Overview cockpit.
2. Show the global AI governance posture.
3. Switch between English and French.
4. Open the Assessment module.
5. Define an AI agent connected to enterprise systems.
6. Calculate the risk score.
7. Save the agent to the inventory.
8. Run prompt injection security tests.
9. Open Compliance and generate the compliance mapping.
10. Explain the OWASP, NIST AI RMF and EU AI Act pre-assessment.
11. Generate the PDF governance report.
12. Show Monitoring and Audit Logs.

## Positioning

AI Governance Sentinel helps answer:

Which AI agents are deployed, what data can they access, what actions can they perform, and what is their governance risk?

## Key message

The real enterprise AI risk is not only the model. The risk comes from data access, connectors, autonomy, weak approvals, poor traceability and missing governance.
